二、C51当中的数据类型

  C语言当中常见的数据类型有以下几类：

char 字符型
int 整型
long 长整型
float 单精度浮点型
double 双精度浮点型
unsigned char 无符号字符型
unsigned int 无符号整型
unsigned long 无符号长整型

  然而在C51当中则还有以下几种常见的数据类型：

sfr 特殊功能寄存器
sfr16 16位特殊功能寄存器
bit 位类型
sbit 可寻址位
